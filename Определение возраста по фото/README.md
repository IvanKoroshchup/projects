#### Описание проекта - Определение возраста по фото

##### Данные
Есть набор фотографий людей с указанием возраста.

##### Задача
Построить модель, которая по фотографии определит приблизительный возраст человека.

##### Используемые библиотеки
Pandas
keras
Matplotlib
Seaborn
компьютерное зрение
машинное обучение
